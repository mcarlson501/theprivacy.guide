#### Why This Matters
Most people use the same password across multiple accounts because remembering dozens of unique passwords is impossible. This is like using the same key for your house, car, office, and safe deposit box. If someone gets that key, they have access to everything.

When companies get hacked (and it happens constantly), criminals test those stolen passwords on other popular sites. If you reuse passwords, one breach can lead to multiple account takeovers. A password manager creates and remembers a unique, strong password for every account automatically.

#### What You'll Accomplish
After completing this task, you will have:
* A unique, strong password for every account you use
* Automatic password filling on all your devices
* Protection against password reuse attacks and data breaches

### Step 1: Choose Your Password Manager
We recommend starting with Bitwarden because it's free, secure, and works everywhere.

1. Go to bitwarden.com and click "Get Started"
2. Create your account with an email you check regularly
3. Create a strong master password you'll remember - this is the ONLY password you'll need to know from now on
4. Write down your master password and store it somewhere safe until you memorize it

**Master Password Tips:** Use a sentence like "My dog Max loves 3 tennis balls!" - easy to remember but hard to guess.

**Alternative Options:**
* 1Password (premium option with excellent interface, $3-4/month)
* Dashlane (more features, premium plans available)

---

### Step 2: Install on All Your Devices
Your password manager needs to work everywhere you use passwords.

1. Download the Bitwarden app on your phone (App Store or Google Play)
2. Install the Bitwarden browser extension on your computer
3. Sign in to both using your master password
4. Test that they sync by adding a test password on one device

**Note:** The browser extension will automatically offer to save and fill passwords as you browse.

---

### Step 3: Import Your Existing Passwords
Don't start from scratch - bring your current passwords into the password manager.

1. In Chrome, go to chrome://settings/passwords and click "Export passwords"
2. In Safari or Firefox, look for "Passwords" in settings and find the export option
3. In your password manager, look for "Import" and upload the exported file
4. Delete the exported file after importing for security

**Note:** Delete the exported file after importing for security.

---

### Step 4: Start Using It (The Most Important Part)

#### Week 1: Just Save Passwords
* Don't change any passwords yet
* When you log into accounts, let the password manager save them
* Get comfortable with the auto-fill feature

#### Week 2: Start Changing Important Passwords
* Begin with your most critical accounts (email, banking)
* Use the password manager to generate new, strong passwords
* Let it save and auto-fill the new passwords

#### Week 3+: Clean Up Everything Else
* Work through your other accounts gradually
* Use the password manager's "Security Checkup" feature to find weak or reused passwords
* Replace them one by one

---

### How to Use It Daily

#### On Your Computer
1. The browser extension will automatically detect login forms
2. Click the extension icon or let it auto-fill
3. For new accounts, let it generate a strong password

#### On Your Phone
1. When you encounter a login screen, tap in the password field
2. Select your password manager from the keyboard suggestions
3. Use Touch ID/Face ID or enter your master password
4. Select the correct account

#### For New Accounts
1. When creating a new account, let the password manager generate the password
2. Don't try to make it "memorable" - that's the password manager's job
3. Save the login immediately

### The Bottom Line

**What You Just Accomplished**
You've eliminated the biggest password security risk - reusing the same password across multiple accounts. Every account now has a unique, strong password that you don't need to remember.

**What to Expect**
Your password manager will offer to save new passwords and fill in existing ones automatically. For the first week, you might need to look up some passwords as you get used to the system, but it quickly becomes effortless.

**Remember:** You've just eliminated one of the easiest ways for criminals to access multiple accounts. This single change dramatically improves your digital security.
